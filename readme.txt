##Calculadora de costos##
Esta calculadora de costos funciona esta pensada para calcular los impuestos de boletas que no traen discriminación de impuestos,
estas traen un SUBTOTAL que es el costo de los productos sin impuestos y un TOTAL que es el costo de los productos más los impuestos, por esto es necesario realizar una operación matemática simple para determinar los impuestos y tazas internas aplicadas por el distribuidor.

SUBTOTAL = 100%
TOTAL = 100% + x (siendo x el porcentaje de impuestos)

regla de 3 simple  TOTAL * 100 / SUBTOTAL = 100% + % de impuestos

ejemplo:
    Subtotal = 2000
    Total = 2500

    2500 * 100 / 2000 = 125
    si al resultado(125) le restamos 100%, nos dá que los impuestos son del 25%

A partir de este punto se realizan otras operaciones para determinar el costo unitario de cada producto.
En el momento que tenemos el costo unitario del producto, hay que tener en cuenta que, en este tipo de voletas, aún no están incluidos los impuestos, es por eso que al moento de calcular el precio de venta se le suma impuestos y luego el porcentaje de ganancia deseado
